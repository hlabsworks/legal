นโยบายความเป็นส่วนตัวของ OffLink

อัปเดตล่าสุด: 22 เมษายน 2026

H Labs Works ("เรา") ดำเนินการแอปพลิเคชันมือถือ "OffLink" ("แอป") นโยบายความเป็นส่วนตัวนี้อธิบายวิธีที่เราเก็บรวบรวม ใช้ และปกป้องข้อมูลของคุณ

1. เกี่ยวกับแอป

OffLink เป็นแอปพลิเคชันโทรศัพท์เสียงแบบกลุ่มที่ใช้เทคโนโลยี Wi-Fi hotspot และ WebRTC โดยไม่ต้องการการเชื่อมต่ออินเทอร์เน็ต ข้อมูลเสียงถูกส่งแบบโลคัลระหว่างอุปกรณ์และไม่ถูกส่งไปยังเซิร์ฟเวอร์ภายนอก

2. ข้อมูลที่เราเก็บรวบรวม

2.1 ข้อมูลที่เก็บรวบรวมโดยตรงจากแอป

ข้อมูลวัตถุประสงค์การจัดเก็บ
ชื่อผู้ใช้ (ไม่บังคับ)ระบุสมาชิกในระหว่างการโทรบนอุปกรณ์เท่านั้น (SharedPreferences)
ข้อมูลกลุ่ม (ชื่อ, SSID, รหัสผ่าน)การตั้งค่าและการเชื่อมต่อการโทรแบบกลุ่มบนอุปกรณ์เท่านั้น (SharedPreferences)
ยอดคงเหลือและประวัติคะแนนการจัดการระบบคะแนนในแอปบนอุปกรณ์เท่านั้น (เข้ารหัส)
ข้อมูลอุปกรณ์ (เวอร์ชัน OS, แบตเตอรี่, หน่วยความจำ)การประเมินการเลือก hostระหว่างเซสชันการโทรเท่านั้น (ไม่คงอยู่)

เกี่ยวกับข้อมูลเสียง: เสียงโทรศัพท์ถูกเข้ารหัสโดยใช้ WebRTC (DTLS-SRTP) และส่งเฉพาะระหว่างอุปกรณ์บนเครือข่ายท้องถิ่น ข้อมูลเสียงไม่ถูกส่งไปยังเซิร์ฟเวอร์ของเราหรือเซิร์ฟเวอร์ภายนอกใดๆ

2.2 ข้อมูลที่เก็บรวบรวมโดยบริการบุคคลที่สาม

แอปใช้บริการบุคคลที่สามต่อไปนี้ ซึ่งอาจเก็บรวบรวมข้อมูลโดยอิสระ

Google AdMob (การโฆษณา)

  • เพื่อการแสดงโฆษณา Google อาจเก็บรวบรวมตัวระบุโฆษณา (Android: AAID) ข้อมูลอุปกรณ์ ตำแหน่งโดยประมาณ และข้อมูลการใช้แอป
  • ข้อมูลที่เก็บรวบรวมอาจถูกใช้เพื่อแสดงโฆษณาที่ปรับแต่งส่วนบุคคล
  • รายละเอียด: นโยบายความเป็นส่วนตัวของ Google

Firebase (การยืนยันตัวตนแบบไม่ระบุชื่อ, Cloud Firestore, Cloud Messaging)

  • แอปใช้การยืนยันตัวตนแบบไม่ระบุชื่อของ Firebase เพื่อระบุเซสชันโดยไม่เชื่อมโยงกับตัวตนส่วนบุคคล ไม่จำเป็นต้องสร้างบัญชีหรือใช้อีเมล/รหัสผ่าน
  • ข้อมูลการเป็นสมาชิกกลุ่ม (ID กลุ่ม, ชื่อที่แสดง, โทเคน FCM) จะถูกเก็บไว้ใน Cloud Firestore เพื่อเปิดใช้งานการแจ้งเตือนแบบ push ข้อมูลจะถูกลบโดยอัตโนมัติเมื่อคุณออกจากกลุ่มหรือถอนการติดตั้งแอป
  • Firebase Cloud Messaging (FCM) ใช้สำหรับส่งการแจ้งเตือนคำเชิญโทรเมื่อแอปอยู่ในพื้นหลัง
  • รายละเอียด: ความเป็นส่วนตัวและความปลอดภัยของ Firebase

Firebase Crashlytics (การรายงานความผิดพลาด)

  • เพื่อปรับปรุงความเสถียรของแอป ข้อมูลความผิดพลาด ข้อผิดพลาดที่ไม่ร้ายแรง และบันทึก breadcrumb จะถูกส่งไปยัง Firebase Crashlytics
  • ตัวระบุผู้ใช้ (ID อุปกรณ์ ฯลฯ) ไม่ถูกส่งเพื่อปกป้องความเป็นส่วนตัวของคุณ Crashlytics ทำงานโดยใช้เฉพาะ Firebase Installation ID
  • ข้อมูลจะถูกส่งและเก็บไว้ในบริการ Firebase ของ Google ไม่รองรับการจัดเก็บข้อมูลในสหภาพยุโรป โดยจะใช้กรอบความเป็นส่วนตัวข้อมูล EU-US (DPF) ของ Google
  • รายละเอียด: ความเป็นส่วนตัวและความปลอดภัยของ Firebase

3. ข้อมูลตำแหน่ง

แอปใช้สิทธิ์ตำแหน่งของ Android (ACCESS_FINE_LOCATION) เพื่อควบคุมฟังก์ชัน Wi-Fi hotspot และรับข้อมูล SSID สิทธิ์นี้ถูกกำหนดโดยระบบปฏิบัติการ Android สำหรับ API ที่เกี่ยวข้องกับ Wi-Fi และไม่ได้ใช้เพื่อติดตาม บันทึก หรือส่งตำแหน่งทางภูมิศาสตร์ของคุณ

4. กล้องถ่ายรูป

แอปใช้สิทธิ์กล้องเพื่อสแกน QR code ภาพที่ถ่ายใช้เพื่อวิเคราะห์ QR code เท่านั้น และไม่ถูกจัดเก็บหรือส่ง

5. ไมโครโฟน

แอปใช้สิทธิ์ไมโครโฟนสำหรับการโทรเสียงแบบกลุ่ม เสียงที่บันทึกถูกส่งแบบเรียลไทม์เฉพาะให้สมาชิกบนเครือข่ายท้องถิ่นเดียวกัน และไม่ถูกบันทึกหรือจัดเก็บบนอุปกรณ์หรือเซิร์ฟเวอร์ใดๆ

6. การแบ่งปันข้อมูลกับบุคคลที่สาม

เราไม่แบ่งปันข้อมูลส่วนบุคคลของคุณกับบุคคลที่สาม ยกเว้นในกรณีต่อไปนี้:

  • ข้อมูลที่อธิบายในส่วนที่ 2.2 จะถูกส่งโดยอัตโนมัติไปยัง Google AdMob เพื่อการแสดงโฆษณา
  • เมื่อกฎหมายกำหนด
  • ด้วยความยินยอมของคุณ

7. การปฏิเสธโฆษณา

หากคุณไม่ต้องการรับโฆษณาที่ปรับแต่งส่วนบุคคล คุณสามารถเปลี่ยนการตั้งค่าได้:

  • Android: การตั้งค่า → Google → โฆษณา → เปิด "ปฏิเสธการปรับแต่งโฆษณา"
  • iOS: การตั้งค่า → ความเป็นส่วนตัวและความปลอดภัย → การติดตาม → ปิด "อนุญาตให้แอปขอติดตาม"

8. การเก็บรักษาและการลบข้อมูล

  • ข้อมูลทั้งหมดที่จัดเก็บโดยตรงจากแอปจะถูกบันทึกเฉพาะบนอุปกรณ์ของคุณ
  • การถอนการติดตั้งแอปจะลบข้อมูลโลคัลทั้งหมด
  • สำหรับข้อมูลที่เก็บรวบรวมโดย Google AdMob โปรดดูนโยบายความเป็นส่วนตัวของ Google

9. ความเป็นส่วนตัวของเด็ก

แอปไม่ได้มีไว้สำหรับเด็กอายุต่ำกว่า 13 ปี และเราไม่เก็บรวบรวมข้อมูลส่วนบุคคลจากเด็กอายุต่ำกว่า 13 ปีโดยเจตนา

10. การเปลี่ยนแปลงนโยบายนี้

เราอาจอัปเดตนโยบายความเป็นส่วนตัวนี้ตามความจำเป็น การเปลี่ยนแปลงที่สำคัญจะถูกแจ้งผ่านการแจ้งเตือนในแอปหรือบนหน้านี้
