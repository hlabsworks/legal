OffLink 개인정보 처리방침

최종 업데이트: 2026년 4월 22일

H Labs Works("저희")는 모바일 애플리케이션 "OffLink"("앱")를 운영합니다. 이 개인정보 처리방침은 저희가 귀하의 정보를 어떻게 수집, 사용, 보호하는지 설명합니다.

1. 앱 소개

OffLink는 Wi-Fi 핫스팟과 WebRTC 기술을 사용하는 그룹 음성통화 애플리케이션으로, 인터넷 연결이 필요하지 않습니다. 음성 데이터는 기기 간에 로컬로 전송되며, 외부 서버로 전송되지 않습니다.

2. 수집하는 정보

2.1 앱이 직접 수집하는 정보

정보목적저장
사용자 이름 (선택)통화 중 구성원 식별기기에만 저장 (SharedPreferences)
그룹 정보 (그룹명, SSID, 비밀번호)그룹 통화 설정 및 연결기기에만 저장 (SharedPreferences)
포인트 잔액 및 내역인앱 포인트 시스템 관리기기에만 저장 (암호화)
기기 정보 (OS 버전, 배터리, 메모리)호스트 선택 평가통화 세션 중에만 (비영구)

음성 데이터 관련: 통화 음성은 WebRTC (DTLS-SRTP)로 암호화되며 로컬 네트워크의 기기 간에만 전송됩니다. 음성 데이터는 저희 서버 또는 외부 서버로 전송되지 않습니다.

2.2 제3자 서비스에서 수집하는 정보

앱은 다음과 같은 제3자 서비스를 이용하며, 이 서비스들은 독립적으로 정보를 수집할 수 있습니다.

Google AdMob (광고)

  • 광고 게재를 위해 Google은 광고 ID (Android: AAID), 기기 정보, 대략적인 위치 및 앱 사용 데이터를 수집할 수 있습니다.
  • 수집된 데이터는 맞춤형 광고 게재에 사용될 수 있습니다.
  • 상세 정보: Google 개인정보 처리방침

Firebase (익명 인증, Cloud Firestore, Cloud Messaging)

  • 앱은 개인 신원과 연결하지 않고 세션을 식별하기 위해 Firebase 익명 인증을 사용합니다. 계정 생성이나 이메일/비밀번호가 필요하지 않습니다.
  • 푸시 알림을 활성화하기 위해 그룹 멤버십 정보(그룹 ID, 표시 이름, FCM 토큰)가 Cloud Firestore에 저장됩니다. 그룹을 나가거나 앱을 삭제하면 데이터가 자동으로 삭제됩니다.
  • Firebase Cloud Messaging (FCM)은 앱이 백그라운드에 있을 때 통화 초대 알림을 전송하는 데 사용됩니다.
  • 상세 정보: Firebase 개인정보 보호 및 보안

Firebase Crashlytics (오류 보고)

  • 앱 안정성 향상을 위해 충돌 정보, 비치명적 오류 및 브레드크럼 로그가 Firebase Crashlytics에 전송됩니다.
  • 귀하의 개인정보를 보호하기 위해 사용자 식별자(기기 ID 등)는 전송되지 않습니다. Crashlytics는 Firebase Installation ID만 사용하여 작동합니다.
  • 데이터는 Google의 Firebase 서비스로 전송되어 저장됩니다. EU 데이터 거주지는 지원되지 않으며, Google의 EU-미국 데이터 개인정보 보호 프레임워크(DPF)가 적용됩니다.
  • 상세 정보: Firebase 개인정보 보호 및 보안

3. 위치 정보

앱은 Wi-Fi 핫스팟 기능 제어 및 SSID 정보 취득을 위해 Android 위치 권한 (ACCESS_FINE_LOCATION)을 사용합니다. 이 권한은 Wi-Fi 관련 API를 위해 Android OS에서 요구하는 것으로, 귀하의 지리적 위치를 추적, 기록, 또는 전송하는 데 사용되지 않습니다.

4. 카메라

앱은 QR 코드 스캔을 위해 카메라 권한을 사용합니다. 촬영된 이미지는 QR 코드 분석에만 사용되며 저장되거나 전송되지 않습니다.

5. 마이크

앱은 그룹 음성 통화를 위해 마이크 권한을 사용합니다. 녹음된 오디오는 동일한 로컬 네트워크의 구성원에게만 실시간으로 전송되며, 어떤 기기나 서버에도 녹음되거나 저장되지 않습니다.

6. 제3자와 데이터 공유

다음의 경우를 제외하고 귀하의 개인 정보를 제3자와 공유하지 않습니다:

  • 2.2항에 명시된 정보는 광고 게재를 위해 Google AdMob에 자동으로 전송됩니다.
  • 법률에 의해 요구되는 경우.
  • 귀하의 동의가 있는 경우.

7. 광고 거부

맞춤형 광고를 받고 싶지 않으시면 설정을 변경할 수 있습니다:

  • Android: 설정 → Google → 광고 → "광고 맞춤설정 사용 중지" 활성화
  • iOS: 설정 → 개인정보 보호 및 보안 → 추적 → "앱이 추적을 요청하도록 허용" 비활성화

8. 데이터 보관 및 삭제

  • 앱이 직접 저장하는 모든 데이터는 귀하의 기기에만 저장됩니다.
  • 앱 삭제 시 모든 로컬 데이터가 삭제됩니다.
  • Google AdMob이 수집한 데이터에 대해서는 Google의 개인정보 처리방침을 참조하세요.

9. 아동 개인정보 보호

앱은 13세 미만 아동을 대상으로 하지 않으며, 13세 미만 아동의 개인 정보를 의도적으로 수집하지 않습니다.

10. 방침 변경

필요에 따라 이 개인정보 처리방침을 업데이트할 수 있습니다. 중요한 변경 사항은 앱 내 알림 또는 이 페이지를 통해 알려드립니다.
