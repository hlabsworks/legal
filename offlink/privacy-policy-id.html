Kebijakan Privasi OffLink

Terakhir diperbarui: 22 April 2026

H Labs Works ("kami") mengoperasikan aplikasi seluler "OffLink" ("Aplikasi"). Kebijakan privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda.

1. Tentang Aplikasi

OffLink adalah aplikasi panggilan suara grup yang menggunakan teknologi hotspot Wi-Fi dan WebRTC tanpa memerlukan koneksi internet. Data suara ditransmisikan secara lokal antar perangkat dan tidak pernah dikirim ke server eksternal.

2. Informasi yang Kami Kumpulkan

2.1 Informasi yang Dikumpulkan Langsung oleh Aplikasi

Tentang data suara: Audio panggilan dienkripsi menggunakan WebRTC (DTLS-SRTP) dan hanya ditransmisikan antar perangkat di jaringan lokal. Data suara tidak pernah dikirim ke server kami atau server eksternal mana pun.

2.2 Informasi yang Dikumpulkan oleh Layanan Pihak Ketiga

Aplikasi menggunakan layanan pihak ketiga berikut, yang mungkin mengumpulkan informasi secara independen.

Google AdMob (Periklanan)

  • Untuk penayangan iklan, Google mungkin mengumpulkan ID iklan (Android: AAID), informasi perangkat, lokasi perkiraan, dan data penggunaan aplikasi.
  • Data yang dikumpulkan dapat digunakan untuk menayangkan iklan yang dipersonalisasi.
  • Detail: Kebijakan Privasi Google

Firebase (Autentikasi Anonim, Cloud Firestore, Cloud Messaging)

  • Aplikasi menggunakan autentikasi anonim Firebase untuk mengidentifikasi sesi tanpa menghubungkannya dengan identitas pribadi. Tidak diperlukan pembuatan akun atau email/kata sandi.
  • Informasi keanggotaan grup (ID grup, nama tampilan, token FCM) disimpan di Cloud Firestore untuk mengaktifkan notifikasi push. Data secara otomatis dihapus ketika Anda meninggalkan grup atau menghapus instalasi Aplikasi.
  • Firebase Cloud Messaging (FCM) digunakan untuk mengirim notifikasi undangan panggilan saat Aplikasi berada di latar belakang.
  • Detail: Privasi dan Keamanan Firebase

Firebase Crashlytics (Laporan Kerusakan)

  • Untuk meningkatkan stabilitas Aplikasi, informasi kerusakan, kesalahan non-fatal, dan log breadcrumb dikirim ke Firebase Crashlytics.
  • Pengidentifikasi pengguna (ID perangkat, dll.) tidak dikirimkan untuk melindungi privasi Anda. Crashlytics beroperasi hanya menggunakan Firebase Installation ID.
  • Data dikirim ke dan disimpan di layanan Firebase Google. Residensi data UE tidak didukung; Kerangka Privasi Data UE-AS (DPF) Google berlaku.
  • Detail: Privasi dan Keamanan Firebase

3. Informasi Lokasi

Aplikasi menggunakan izin lokasi Android (ACCESS_FINE_LOCATION) untuk mengontrol fungsi hotspot Wi-Fi dan mengambil informasi SSID. Izin ini diperlukan oleh OS Android untuk API terkait Wi-Fi dan tidak digunakan untuk melacak, merekam, atau mentransmisikan lokasi geografis Anda.

4. Kamera

Aplikasi menggunakan izin kamera untuk memindai kode QR. Gambar yang ditangkap hanya digunakan untuk analisis kode QR dan tidak disimpan atau ditransmisikan.

5. Mikrofon

Aplikasi menggunakan izin mikrofon untuk panggilan suara grup. Audio yang direkam ditransmisikan secara real-time hanya kepada anggota di jaringan lokal yang sama dan tidak pernah direkam atau disimpan di perangkat atau server mana pun.

6. Berbagi Data dengan Pihak Ketiga

Kami tidak membagikan informasi pribadi Anda dengan pihak ketiga kecuali dalam kasus berikut:

  • Informasi yang dijelaskan di Bagian 2.2 secara otomatis dikirim ke Google AdMob untuk penayangan iklan.
  • Ketika diwajibkan oleh hukum.
  • Dengan persetujuan Anda.

7. Opt-Out Iklan

Jika Anda tidak ingin menerima iklan yang dipersonalisasi, Anda dapat mengubah pengaturan Anda:

  • Android: Pengaturan → Google → Iklan → Aktifkan "Nonaktifkan Personalisasi Iklan"
  • iOS: Pengaturan → Privasi & Keamanan → Pelacakan → Nonaktifkan "Izinkan Aplikasi Meminta untuk Melacak"

8. Penyimpanan dan Penghapusan Data

  • Semua data yang disimpan langsung oleh Aplikasi hanya tersimpan di perangkat Anda.
  • Menghapus instalasi Aplikasi akan menghapus semua data lokal.
  • Untuk data yang dikumpulkan oleh Google AdMob, silakan lihat kebijakan privasi Google.

9. Privasi Anak-anak

Aplikasi tidak ditujukan untuk anak-anak di bawah 13 tahun dan kami tidak secara sengaja mengumpulkan informasi pribadi dari anak-anak di bawah 13 tahun.

10. Perubahan Kebijakan Ini

Kami dapat memperbarui kebijakan privasi ini sesuai kebutuhan. Perubahan signifikan akan dikomunikasikan melalui notifikasi dalam aplikasi atau di halaman ini.
