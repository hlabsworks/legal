OffLink 隐私政策

最后更新：2026年4月22日

H Labs Works（"我们"）运营移动应用程序"OffLink"（"本应用"）。本隐私政策说明我们如何收集、使用和保护您的信息。

1. 关于本应用

OffLink 是一款使用 Wi-Fi 热点和 WebRTC 技术的群组语音通话应用，无需互联网连接。语音数据在设备之间本地传输，从不发送到外部服务器。

2. 我们收集的信息

2.1 本应用直接收集的信息

信息用途存储
用户名（可选）通话期间识别成员仅在设备上（SharedPreferences）
群组信息（名称、SSID、密码）群组通话设置和连接仅在设备上（SharedPreferences）
积分余额和历史应用内积分系统管理仅在设备上（已加密）
设备信息（OS版本、电池、内存）主机选择评估仅在通话会话期间（非持久）

关于语音数据：通话音频使用 WebRTC（DTLS-SRTP）加密，仅在本地网络上的设备之间传输。语音数据从不发送到我们的服务器或任何外部服务器。

2.2 第三方服务收集的信息

本应用使用以下第三方服务，这些服务可能独立收集信息。

Google AdMob（广告）

  • 为了投放广告，Google 可能收集广告标识符（Android: AAID）、设备信息、大致位置和应用使用数据。
  • 收集的数据可能用于展示个性化广告。
  • 详情：Google 隐私政策

Firebase（匿名身份验证、Cloud Firestore、Cloud Messaging）

  • 本应用使用 Firebase 匿名身份验证来识别会话，而不与个人身份关联。无需创建账户或输入邮箱/密码。
  • 群组成员信息（群组 ID、显示名称、FCM 令牌）存储在 Cloud Firestore 中以启用推送通知。当您离开群组或卸载本应用时，数据将自动删除。
  • Firebase Cloud Messaging (FCM) 用于在本应用处于后台时发送通话邀请通知。
  • 详情：Firebase 隐私和安全

Firebase Crashlytics（崩溃报告）

  • 为提高应用稳定性，崩溃信息、非致命错误和面包屑日志将发送到 Firebase Crashlytics。
  • 为保护您的隐私，不传输用户标识符（设备 ID 等）。Crashlytics 仅使用 Firebase Installation ID 运行。
  • 数据发送至 Google Firebase 服务并存储。不支持欧盟数据驻留；适用 Google 的欧盟-美国数据隐私框架（DPF）。
  • 详情：Firebase 隐私和安全

3. 位置信息

本应用使用 Android 位置权限（ACCESS_FINE_LOCATION）来控制 Wi-Fi 热点功能并获取 SSID 信息。此权限由 Android 操作系统对 Wi-Fi 相关 API 要求，不用于追踪、记录或传输您的地理位置。

4. 相机

本应用使用相机权限扫描二维码。拍摄的图像仅用于二维码分析，不会被存储或传输。

5. 麦克风

本应用使用麦克风权限进行群组语音通话。录制的音频仅实时传输给同一本地网络上的成员，不会在任何设备或服务器上录制或存储。

6. 与第三方共享数据

除以下情况外，我们不与第三方共享您的个人信息：

  • 第2.2节中描述的信息会自动传输到 Google AdMob 用于广告投放。
  • 法律要求时。
  • 经您同意。

7. 广告退出

如果您不想接收个性化广告，可以更改设置：

  • Android：设置 → Google → 广告 → 开启"停用广告个性化"
  • iOS：设置 → 隐私与安全性 → 追踪 → 关闭"允许应用请求追踪"

8. 数据保留和删除

  • 本应用直接存储的所有数据仅保存在您的设备上。
  • 卸载本应用将删除所有本地数据。
  • 对于 Google AdMob 收集的数据，请参阅 Google 的隐私政策。

9. 儿童隐私

本应用不面向13岁以下儿童，我们不会故意收集13岁以下儿童的个人信息。

10. 本政策的变更

我们可能会根据需要更新本隐私政策。重大变更将通过应用内通知或本页面告知您。
