Política de Privacidade do OffLink

Última atualização: 22 de abril de 2026

H Labs Works ("nós") opera o aplicativo móvel "OffLink" ("o App"). Esta política de privacidade descreve como coletamos, usamos e protegemos suas informações.

1. Sobre o App

OffLink é um aplicativo de chamadas de voz em grupo que utiliza tecnologia de hotspot Wi-Fi e WebRTC sem necessidade de conexão à internet. Os dados de voz são transmitidos localmente entre dispositivos e nunca são enviados a servidores externos.

2. Informações que coletamos

2.1 Informações coletadas diretamente pelo App

InformaçãoFinalidadeArmazenamento
Nome de usuário (opcional)Identificação de membros durante chamadasApenas no dispositivo (SharedPreferences)
Informações do grupo (nome, SSID, senha)Configuração e conexão de chamadas em grupoApenas no dispositivo (SharedPreferences)

Informações do dispositivo (versão do OS, bateria, memória)Avaliação de seleção de hostApenas durante a sessão de chamada (não persistente)

Sobre dados de voz: O áudio das chamadas é criptografado usando WebRTC (DTLS-SRTP) e transmitido apenas entre dispositivos na rede local. Os dados de voz nunca são enviados aos nossos servidores ou a qualquer servidor externo.

2.2 Informações coletadas por serviços de terceiros

O App utiliza os seguintes serviços de terceiros, que podem coletar informações de forma independente.

Google AdMob (Publicidade)

  • Para exibição de anúncios, o Google pode coletar identificadores de publicidade (Android: AAID), informações do dispositivo, localização aproximada e dados de uso do app.
  • Os dados coletados podem ser usados para exibir anúncios personalizados.
  • Detalhes: Política de Privacidade do Google

Firebase (Autenticação anônima, Cloud Firestore, Cloud Messaging)

  • O App utiliza a autenticação anônima do Firebase para identificar sessões sem vinculá-las à identidade pessoal. Não é necessário criar conta ou usar email/senha.
  • Informações de associação ao grupo (ID do grupo, nome de exibição, token FCM) são armazenadas no Cloud Firestore para possibilitar notificações push. Os dados são excluídos automaticamente quando você sai de um grupo ou desinstala o App.
  • O Firebase Cloud Messaging (FCM) é usado para enviar notificações de convite de chamada quando o App está em segundo plano.
  • Detalhes: Privacidade e segurança do Firebase

Firebase Crashlytics (Relatórios de falha)

  • Para melhorar a estabilidade do App, informações de falhas, erros não fatais e registros de navegação são enviados ao Firebase Crashlytics.
  • Identificadores de usuário (IDs de dispositivo, etc.) não são transmitidos para proteger sua privacidade. O Crashlytics opera usando apenas o ID de instalação do Firebase.
  • Os dados são enviados e armazenados nos serviços Firebase do Google. A residência de dados na UE não é suportada; aplica-se o Framework de Privacidade de Dados UE-EUA (DPF) do Google.
  • Detalhes: Privacidade e segurança do Firebase

3. Informações de localização

O App utiliza a permissão de localização do Android (ACCESS_FINE_LOCATION) para controlar a funcionalidade de hotspot Wi-Fi e obter informações de SSID. Essa permissão é exigida pelo sistema operacional Android para APIs relacionadas ao Wi-Fi e não é usada para rastrear, registrar ou transmitir sua localização geográfica.

4. Câmera

O App usa a permissão de câmera para escanear códigos QR. As imagens capturadas são usadas apenas para análise de código QR e não são armazenadas ou transmitidas.

5. Microfone

O App usa a permissão de microfone para chamadas de voz em grupo. O áudio gravado é transmitido em tempo real apenas para membros na mesma rede local e nunca é gravado ou armazenado em qualquer dispositivo ou servidor.

6. Compartilhamento de dados com terceiros

Não compartilhamos suas informações pessoais com terceiros, exceto nos seguintes casos:

  • As informações descritas na Seção 2.2 são automaticamente transmitidas ao Google AdMob para entrega de anúncios.
  • Quando exigido por lei.
  • Com seu consentimento.

7. Opt-out de publicidade

Se você não quiser receber anúncios personalizados, pode alterar suas configurações:

  • Android: Configurações → Google → Anúncios → Ativar "Cancelar personalização de anúncios"
  • iOS: Ajustes → Privacidade e Segurança → Rastreamento → Desativar "Permitir que apps solicitem rastreamento"

8. Retenção e exclusão de dados

  • Todos os dados armazenados diretamente pelo App são salvos apenas no seu dispositivo.
  • Desinstalar o App excluirá todos os dados locais.
  • Para dados coletados pelo Google AdMob, consulte a política de privacidade do Google.

9. Privacidade de crianças

O App não é destinado a crianças menores de 13 anos e não coletamos intencionalmente informações pessoais de crianças menores de 13 anos.

10. Alterações nesta política

Podemos atualizar esta política de privacidade conforme necessário. Mudanças significativas serão comunicadas por meio de notificações no app ou nesta página.
